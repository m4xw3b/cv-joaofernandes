# 💾 Currículo Retro - Windows XP Edition

Bem-vindo ao repositório do meu currículo interativo! 
Este projeto é uma viagem no tempo até ao início dos anos 2000, apresentando o meu percurso profissional e académico através da interface clássica do **Windows XP**.

🔗 **[Clica aqui para ver o Currículo Online] (https://github.com/m4xw3b/cv-joaofernandes)**

## 🚀 Funcionalidades

O ambiente de trabalho simula um verdadeiro sistema operativo vintage, incluindo janelas interativas e arrastáveis:

*   **curriculo_joao.exe:** O meu perfil, experiência profissional e formação.
*   **Wintech (Internet Explorer):** Um atalho direto para o meu portal de informação tecnológica.
*   **Contactos (Bloco de Notas):** Os meus contactos apresentados no clássico formato .txt.
*   **Minesweeper:** Uma versão totalmente jogável do clássico jogo das Minas, desenvolvida do zero em JavaScript.

## 🛠️ Tecnologias Utilizadas

Este projeto foi construído de forma "Vanilla" (sem frameworks complexas) para garantir máxima leveza e compatibilidade:

*   **HTML5 & CSS3:** Estrutura e estilização.
*   **JavaScript:** Lógica das janelas (drag & drop) e motor do jogo Minesweeper.
*   **[XP.css](https://botoxparty.github.io/XP.css/):** Biblioteca CSS open-source utilizada para recriar com precisão os botões, barras de título e janelas clássicas.
